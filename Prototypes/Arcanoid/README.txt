# Cosmic Breaker

Космическая аркада с разбиванием блоков.

## Управление:
- ← → или A D - движение платформы
- R - рестарт игры (после победы или проигрыша)
- ESC - выход

## Особенности:
- Разноцветные блоки
- Система жизней
- Счет очков
- Автоматическое создание новых блоков при рестарте

## Запуск:
python main.py

# Анализ Core Loop — Minecraft

**Описание:** Minecraft — песочница, где игрок исследует, добывает ресурсы и строит.

**Core Loop:**
- Action: добыча ресурсов
- Reward: материалы (дерево, руда, еда)
- Progression: крафт предметов, строительство, исследование новых биомов

**Схема:**
Добыча → Крафт → Улучшение → Исследование → Добыча снова

**Вывод:** Core loop работает, потому что игрок всегда получает краткосрочные награды (материалы) и долгосренный прогресс (постройка базы, исследование мира).
